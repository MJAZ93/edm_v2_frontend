# Sistema EDM Vandalizações - Descrição para Apresentação

## Visão Geral do Sistema

O **Sistema de Analise de dados da EDM** é uma plataforma web abrangente desenvolvida para a gestão integrada de incidentes de vandalismo e infrações. Trata-se de uma solução completa que permite o controlo, monitorização e análise de ocorrências de vandalismo, proporcionando uma gestão eficiente desde o registo até à resolução das situações.

## Módulos Principais do Sistema

### 1. Gestão de Ocorrências
**Funcionalidade Central:** Registo e acompanhamento de incidentes de vandalismo
- Criação, edição e visualização detalhada de ocorrências
- Categorização por tipos de infração
- Anexação de evidências e documentação
- Workflow de estados da ocorrência

### 2. Gestão de Infrações
**Controlo de Violações:** Tipificação e gestão de infrações
- Catálogo de tipos de infração configuráveis
- Setores de infração para organização territorial
- Histórico e rastreabilidade de infrações
- Associação com legislação aplicável

### 3. Gestão de Infractores
**Registo de Entidades:** Pessoas e organizações envolvidas
- Base de dados de infractores (pessoas físicas/jurídicas)
- Histórico de infrações por infractor
- Dados de contacto e identificação
- Análise de padrões comportamentais

### 4. Gestão de Sucataris
**Controlo de Materiais:** Monitorização de sucatarias e materiais
- Registo de sucatarias com geolocalização
- Integração com sistema de mapas
- Inventário de materiais recuperados
- Rastreabilidade de origem dos materiais

### 5. Gestão de Clientes
**Relacionamento Comercial:** Gestão de entidades cliente
- Base de dados de clientes
- Ações associadas a clientes
- Dashboard específico de clientes
- Análise de inspeções por cliente

### 6. Gestão de Ações
**Planeamento e Execução:** Ações corretivas e preventivas
- Planeamento de ações
- Acompanhamento de execução
- Associação com ocorrências e clientes
- Tipos de ação configuráveis
